# UI_MOCKUP

A UI mockup project for designing and prototyping user interfaces.

## Project Structure

```
UI_MOCKUP/
├── index.html        # Main entry point
├── css/
│   └── styles.css    # Main stylesheet
├── js/
│   └── main.js       # Main JavaScript file
└── assets/           # Images, fonts, and other static assets
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/kaihsinlin-dotcom/UI_MOCKUP.git
   cd UI_MOCKUP
   ```

2. Open `index.html` in your browser to view the mockup.

## Development

- Edit `index.html` to update the page structure.
- Edit `css/styles.css` to update the styles.
- Edit `js/main.js` to add interactivity.
